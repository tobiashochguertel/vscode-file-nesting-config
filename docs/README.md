# Personal File Nesting Config for VS Code

This project provides a **personal** VS Code file nesting configuration for
[Tobias Hochgürtel](https://github.com/tobiashochguertel) — and anyone else who finds it useful.

It is built on top of the community-maintained
[antfu/vscode-file-nesting-config](https://github.com/antfu/vscode-file-nesting-config) and extends it
with custom patterns for tools like `mise` and `Taskfile`.

The `README.md` in the root of this repo is **auto-generated** — never edit it directly.
To add or change patterns, edit [`personal-patterns.yaml`](../personal-patterns.yaml).
